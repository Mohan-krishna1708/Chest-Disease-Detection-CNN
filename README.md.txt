# Chest Disease Detection using CNN

This project uses a Convolutional Neural Network (CNN) to detect chest diseases from X-ray images.

## Dataset
The dataset used is the NIH Chest X-ray dataset:
https://www.kaggle.com/datasets/nih-chest-xrays/data

Only the dataset link is provided here. The full dataset is not uploaded to this repository.

## Features
- Image classification using CNN
- Detects classes: Normal, Pneumonia, Fibrosis, Inflammation
- Compares CNN with Random Forest and Decision Tree
- Shows accuracy, precision, recall, F1-score and confusion matrix

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy, Pandas, Matplotlib, Scikit-learn

## How to Run

1. Clone this repository
2. Install dependencies:
   pip install -r requirements.txt
3. Open the notebook:
   jupyter notebook ChestDisease.ipynb
4. Run all cells
